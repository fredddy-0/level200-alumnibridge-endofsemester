# level200-InsightHub-endofsemester
End of Semester project - Frederick Kwabena Twumasi

INSIGHTHUB is a centralized digital platform designed to store, manage, and explore student research papers and capstone projects. It allows students to submit their work, and users to discover and engage with academic projects.

Features
User Authentication
User registration and login system
Secure password storage using hashing
JWT-based authentication
Project Submission
